# 🚀 FreelaWise

## 🇵🇹 Sobre o projeto

O **FreelaWise** é uma aplicação SaaS full-stack desenvolvida para ajudar freelancers a gerenciar seus rendimentos, despesas e estimativas de impostos de forma simples e eficiente.

O sistema permite visualizar quanto o usuário pode gastar com segurança após considerar impostos e custos operacionais.

Este projeto foi desenvolvido com foco em boas práticas de engenharia de software, arquitetura limpa e tecnologias modernas utilizadas no mercado.

---

## EN About the project

**FreelaWise** is a full-stack SaaS application designed to help freelancers manage income, expenses, and tax estimations efficiently.

It provides insights into how much money can be safely spent after accounting for taxes and operational costs.

This project focuses on clean architecture, best practices, and modern technologies used in real-world applications.

---

# 🧠 Architecture

- Hexagonal Architecture (Ports & Adapters)
- Separation of concerns (Domain, Application, Infrastructure)
- Scalable and maintainable structure
